  TUI (Text User Interface) Approach

  ncurses

  - What it is: Low-level C library for terminal control (1993, still the
  standard)
  - How it works: Direct terminal manipulation - position cursor, draw
  characters at specific coordinates, handle input events
  - Capabilities: Full control - sticky headers, split panes, real-time updates,
   mouse support
  - Node.js bindings: blessed, node-ncurses

  Example capabilities:
  // Pseudo-code with blessed (ncurses wrapper)
  const header = blessed.box({
    top: 0,
    height: 3,
    content: 'HEAD~1 | 5 files changed +120 -45',
    style: { bg: '#30292f' }
  });

  const diffView = blessed.list({
    top: 3,
    scrollable: true,
    keys: true,
    vi: true // Vim keybindings
  });

  ink

  - What it is: React for terminals (2017, by Vercel)
  - How it works: Component-based, uses React reconciliation, renders to
  terminal
  - Architecture: Built on yoga (flexbox layout) + lower-level terminal libs
  - Best for: Interactive CLIs with forms, spinners, progress bars

  Relationship to ncurses:
  - ink is higher-level than ncurses
  - ncurses gives you pixel-by-pixel (character-by-character) control
  - ink gives you components and layout

  Example:
  const DiffViewer = () => {

  What's New

  Two Modes:
  bonsai HEAD~1         # TUI mode (default) - interactive review
  bonsai HEAD~1 --dump  # Dump mode - static output (what you had before)

  TUI Features Implemented

  ✅ Sticky Header
  - Shows commit range, branch name, file count
  - Review progress: "3/5 files reviewed ✓"
  - Current file with status indicator (✓ = reviewed, ○ = not reviewed)
  - Never scrolls away - always know where you are

  ✅ Review State Tracking
  - Press r to mark current file as reviewed
  - State saves to .bonsai-review-{branch}.json (gitignored)
  - Per-branch tracking - each branch has independent state
  - Resume reviews anytime - restores your scroll position

  ✅ File Navigation
  - n - Next file
  - p - Previous file
  - f - Toggle file list (shows all files with ✓/○ indicators)
  - Arrow keys + Enter to select file in list
  - Auto-saves scroll position when switching files

  ✅ Vim Keybindings
  - j/k - Scroll up/down
  - d/u - Half page scroll
  - g/G - Jump to top/bottom
  - Mouse scrolling works too
  - q or Ctrl+C - Quit

  ✅ Your Custom Colors
  - Same #30292f background
  - Same #00635D teal and #8F000A red accents
  - All styling preserved

  File Structure

  src/
    cli.js        - Routes to TUI or dump mode
    tui-mode.js   - Interactive TUI (blessed-based)
    dump-mode.js  - Static output (your original code)
    index.js      - Entry point
